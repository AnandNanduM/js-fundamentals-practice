Built-in Objects Examples

JSON Methods (JSON.parse and JSON.stringify) 
            // JSON methods example
            const jsonString = '{"name": "John", "age": 30, "city": "New York"}';
    
            // JSON.parse() to parse JSON string to JavaScript object
            const jsonObject = JSON.parse(jsonString);
            console.log('Parsed JSON:', jsonObject);
    
            // JSON.stringify() to convert JavaScript object to JSON string
            const jsonStringified = JSON.stringify(jsonObject);
            console.log('Stringified JSON:', jsonStringified);
    
            // Display JSON examples in HTML
            document.getElementById('jsonExample').textContent = `Parsed JSON:\n${JSON.stringify(jsonObject, null, 4)}\nStringified JSON: ${jsonStringified}`;
Map Example 
            // Map example
            const mapExample = new Map();
            mapExample.set('key1', 'value1');
            mapExample.set('key2', 'value2');
    
            // Display Map example in HTML
            document.getElementById('mapExample').textContent = `Map Example:\n${JSON.stringify([...mapExample], null, 4)}`;
Set Example 
            // Set example
            const setExample = new Set();
            setExample.add('value1');
            setExample.add('value2');
            setExample.add('value1'); // Sets do not allow duplicate values
    
            // Display Set example in HTML
            document.getElementById('setExample').textContent = `Set Example:\n${JSON.stringify([...setExample], null, 4)}`;

JavaScript Basics

Welcome to the JavaScript Basics page! This page demonstrates some fundamental concepts of JavaScript programming language in a beginner-friendly manner.

Variable Declaration

In JavaScript, you can declare variables using var, let, and const keywords.

Let's declare some variables:

    // Variable Declaration
    var myVar = 'Hello'; // old way, function-scoped
    let myLet = 'World'; // block-scoped, can be reassigned
    const myConst = '!';

    console.log('myVar:', myVar);
    console.log('myLet:', myLet);
    console.log('myConst:', myConst);
Data Types

JavaScript supports various data types including numbers, strings, booleans, null, undefined, and symbols.

Let's define some variables with different data types:

    // Data Types
    var num = 10; // Number
    var str = 'Hello'; // String
    var bool = true; // Boolean
    var nul = null; // Null
    var und = undefined; // Undefined
    var sym = Symbol('foo'); // Symbol

    console.log('num:', num);
    console.log('str:', str);
    console.log('bool:', bool);
    console.log('nul:', nul);
    console.log('und:', und);
    console.log('sym:', sym);
Operations

JavaScript supports various operations such as arithmetic, comparison, and logical operations.

Let's perform some operations:

    
    // Operations
    // Arithmetic
    var addition = 5 + 3; // 8
    var subtraction = 10 - 5; // 5
    var multiplication = 4 * 2; // 8
    var division = 20 / 4; // 5
    var modulus = 10 % 3; // 1

    console.log('addition:', addition);
    console.log('subtraction:', subtraction);
    console.log('multiplication:', multiplication);
    console.log('division:', division);
    console.log('modulus:', modulus);

    // Comparison
    var equal = 5 === '5'; // false
    var notEqual = 5 !== '5'; // true
    var greaterThan = 10 > 5; // true
    var lessThan = 5 < 10; // true
    var greaterThanOrEqual = 10 >= 10; // true
    var lessThanOrEqual = 5 <= 10; // true

    console.log('equal:', equal);
    console.log('notEqual:', notEqual);
    console.log('greaterThan:', greaterThan);
    console.log('lessThan:', lessThan);
    console.log('greaterThanOrEqual:', greaterThanOrEqual);
    console.log('lessThanOrEqual:', lessThanOrEqual);

    // Logical
    var and = true && false; // false
    var or = true || false; // true
    var not = !true; // false

    console.log('and:', and);
    console.log('or:', or);
    console.log('not:', not);
Conditional Statements

JavaScript provides conditional statements such as if, else if, and else to execute different code blocks based on certain conditions.

Let's see some examples:

    // Conditional Statements
    var age = 20;

    if (age < 18) {
      console.log('You are a minor.');
    } else if (age >= 18 && age < 60) {
      console.log('You are an adult.');
    } else {
      console.log('You are a senior citizen.');
    }
Iterations or Loops

JavaScript provides different types of loops such as for, while, do...while.

Let's see some examples:

    // For loop
    console.log('For loop:');
    for (var i = 1; i <= 5; i++) {
      console.log(i);
    }

    // While loop with break and continue
    console.log('While loop with break and continue:');
    var j = 1;
    while (j <= 10) {
      if (j === 5) {
        break; // Exit loop when j is 5
      }
      if (j % 2 === 0) {
        j++; // Skip even numbers
        continue; // Skip current iteration and move to next iteration
      }
      console.log(j);
      j++;
    }
